## ğŸ“œ Intro

**Hero** is a browser window emulator for Rust-based WASM frameworks like **Yew**, **Leptos**, and **Dioxus**. Designed to wrap embedded content in a mock browser frame, it offers complete control over styling, layout, and interactions.

## ğŸ¤” Why Use Hero?

1. **ğŸ–¼ï¸ Realistic Browser Look**: Simulates a browser window with address bar, controls (close, minimize, maximize), and content area.

1. **ğŸ”© Highly Configurable**: Toggle visibility, customize styles, and control behavior with dozens of props.

1. **ğŸ“Ÿ Accessible by Default**: Comes with semantic roles and ARIA attributes for screen readers and keyboard users.

1. **âš¡ Interactive Hooks**: Emit callbacks on user actions like URL change, hover, click, blur, focus, and keyboard events.

1. **ğŸ›ï¸ Theming-Ready and Style-Agnostic**: Comes with sensible defaults but is easily styled via Tailwind, CSS classes, or inline styles.
